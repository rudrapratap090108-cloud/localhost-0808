# Background music

Place your kids-style background music file here as **`bg-music.mp3`**.

- File must be named exactly `bg-music.mp3` (or update `MUSIC_SRC` in `src/components/FloatingActions.tsx`).
- Keep it under ~3 MB and set it to a gentle loopable track.
- The floating yellow speaker button on the homepage plays/pauses it.
  It starts **muted** because browsers block autoplay with sound.
